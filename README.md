
#  Medi-Quick: Essential Medicine Finder

#  Project Overview
**Medi-Quick** is a web application designed to solve the real-life problem of fragmented medical information. It allows users to quickly search for medications, understand their purposes, and filter them by category using real-time data. This project demonstrates proficiency in  JavaScript, API integration via Fetch, and Responsive UI design.

##  API Integration
* **Source:** [openFDA Drug Label API](https://open.fda.gov/apis/drug/).
* **Endpoint:** `https://api.fda.gov/drug/label.json?limit=50`
* **Purpose:** To fetch dynamic, verified data including brand names, generic names, and indications for use.

##  Key Features (Milestone 3 Requirements)
As per the project guidelines, all data manipulation is performed using **Array Higher-Order Functions (HOFs)** like `.filter()`, `.map()`, and `.sort()`. No traditional `for` or `while` loops are used.
